## About The Project


The Project is aimed at developing the warehouse console application allowing its user:
* At the first step creating the object with initial information on a new storage parcel, the scope of which depends on whether the bailor is a legal entity or an individual registered or not registered as a private entrepreneur. All the new objects are added to the relevant Array Lists for legal entities and individuals separately.
* Then, via references to the relevant information items in the relevant Array Lists, a user is able to use the menu and by making the suggested choise is able to:
** Find existing parcel with its ID number
** Calculate the period of storage for specified / existing parcel
** Calculate an amount of the storage charge for existing parcel
** Exit from the application")
* The application contains the information notes helping a user to make a choise and indicating the mistakes made, if any.
